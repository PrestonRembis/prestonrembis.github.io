Rust Google TUI

This was a class project where we could develop anything we wanted over the semester. We developed a text-based user interface (TUI) that interacted with the Google API. You can see your most recent emails, send emails, do Google searches from the TUI, and more. I mainly worked on the frontend which was coded in Rust, which was new to me. Overall I am very proud of this project as in my experience Rust is a quite stringent and really pushes you to think. The group was able to produce a useful tool which required weekly meetings and working on one repository in Git which gave us great workflow experience.
